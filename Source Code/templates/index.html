Basketball Statistics Database

League Histories

Represents the complete history of a basketball organization. Allows the user to view, add, delete, and update leagues.

E.g. The National Basketball Association (NBA).

Seasons

Represents a single season of a basketball organization. Allows the user to view, add, delete, and update seasons.

E.g. The 1995-1996 NBA Season

Teams

Represents an individual team within a single season. Allows the user to view, add, delete, and update teams.

E.g. The 1995-1996 Chicago Bulls.

Players

Represents an individual player of a single team (within a single season). Allows the user to view, add, delete, and update players.

E.g. Michael Jordan of the 1995-1996 Chicago Bulls.

Coaches

Represents an individual coach of a single team (within a single season). Allows the user to view, add, delete, and update coaches.

E.g. Phil Jackson of the 1995-1996 Chicago Bulls.

Player Team Relations

Represents the playing relationship between an individual player and a single team (within a single season). Allows the user to view, add, delete, and update player team relations.

E.g. Michael Jordan's playing relationship with the 1995-1996 Chicago Bulls.

Coach Team Relations

Represents the coaching relationship between an individual coach and a single team (within a single season). Allows the user to view, add, delete, and update coach team relations.

E.g. Phil Jackson's coaching relationship with the 1995-1996 Chicago Bulls.
